# simplifa11y

> simple a11y-ready react component library

## Install

```bash
yarn add @pipeDr3am/simplifa11y -s
```

## Usage

```jsx
import React from 'react'

import MyComponent from 'simplifa11y'
import 'simplifa11y/dist/index.css'

const Example = () => {
  return <MyComponent />
}
```
